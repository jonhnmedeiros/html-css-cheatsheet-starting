Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<th> Table Header The set of rows defining the column headers in a table.
<tfoot> Table Head The set of rows defining the footer in a table.
<caption> Table Caption The caption for a table.
<col> Table Column The column properties for each column within a colgroup.
<colgroup> Table Column Group The group of columns within a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
scope Scope Defines the scope of a th element.
headers Headers Defines the header cells a cell is related to.
border Border Defines the border width of a table.
cellpadding Cell Padding Defines the padding within a cell.
cellspacing Cell Spacing Defines the spacing between cells.
width Width Defines the width of a table.
height Height Defines the height of a table.
align Align Defines the alignment of a table.
valign Vertical Align Defines the vertical alignment of a table.
bgcolor Background Color Defines the background color of a table.
background Background Image Defines the background image of a table.
summary Summary Defines a summary of a table.
frame Frame Defines the frame around a table.
rules Rules Defines the rules between rows and columns in a table.
class Class Defines a class for a table.
id ID Defines an ID for a table.
style Style Defines an inline style for a table.
